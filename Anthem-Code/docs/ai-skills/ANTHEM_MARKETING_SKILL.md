# Anthem Marketing Skill

## Purpose

ใช้เมื่อวางแผน:

- demo review
- soft launch
- creator seeding
- referral campaign
- public launch
- content strategy

## Positioning

Anthem คือ creator community ไม่ใช่เว็บประกาศงานทั่วไป

Core message:

> ลงผลงานแรก สร้างตัวตน หาโอกาส และชวนเพื่อนเข้าชุมชนครีเอเตอร์ไทย

## Launch Phases

### Phase 1 - Closed Demo Review

Goal:

- หาข้อสะดุด UX ก่อนเปิดคนเยอะ
- เก็บ testimonial/quote
- วัด first post friction

Audience:

- 20-50 reviewer
- designer, student, freelancer, client, community reviewer

Assets:

- demo URL
- usability checklist PDF
- feedback form
- reviewer instructions

### Phase 2 - Soft Launch

Goal:

- seed creator
- เปิด referral แบบคุมงบ
- วัด active user ไม่ใช่ signup

Campaign:

- "ลงงานแรก รับ PX"
- "ชวนเพื่อน รับ reward เมื่อเพื่อนโพสต์งานแรก"
- weekly showcase

### Phase 3 - Creator Seeding

Goal:

- มีผลงานก่อนเปิดกว้าง
- ทำให้ feed ดูมีชีวิต

Targets:

- 30-100 creator แรก
- art/design students
- junior freelancers
- micro creator

Tactics:

- DM creator พร้อม template
- feature งานดีบน social
- weekly challenge
- creator spotlight

### Phase 4 - Public Launch

Goal:

- เปิดกว้างอย่างมี message ชัด
- ใช้ referral เป็น growth loop
- เริ่ม test ads งบน้อย

Channels:

- TikTok
- Facebook groups
- X
- Instagram
- university/community pages
- creator newsletters

## Content Pillars

1. Showcase ผลงาน
2. Creator journey / before-after portfolio
3. Tutorial "ลงงานแรกใน 3 นาที"
4. Referral and reward education
5. Jobs/opportunity
6. Community challenge
7. Trust/safety/IP education

## KPIs

Primary:

- signup
- first post rate
- referral activation rate
- creator 7-day retention
- project/post count

Secondary:

- profile completion
- hire/collab request
- comments/follows
- report rate
- feedback volume

Avoid vanity-only:

- signup จำนวนมากแต่ไม่มี first post
- referral click เยอะแต่ไม่ active
- pageview สูงแต่ไม่มี creator action

## Referral Campaign Guardrails

- pay per performance only
- referrer paid after first meaningful action
- campaign budget cap before scale
- fraud review for suspicious spikes
- clear copy about cashout threshold

## Messaging Examples

Short:

- "พื้นที่โชว์งานของครีเอเตอร์ไทย"
- "ลงงานแรก สร้างตัวตน และให้คนเจอคุณ"
- "ชวนเพื่อนเข้าชุมชน รับ reward เมื่อเพื่อนเริ่มใช้งานจริง"

Avoid:

- "สมัครแล้วได้เงินทันที"
- "หาเงินฟรี"
- "รวยจากชวนเพื่อน"

## Launch Readiness Criteria

ก่อนเปิด public:

- [ ] usability blockers fixed
- [ ] feed มีผลงานพอ
- [ ] first post flow smooth
- [ ] referral terms clear
- [ ] report/feedback works
- [ ] demo/production URLs stable
- [ ] content calendar 30 วันพร้อม

## 30-Day Content Rhythm

Weekly:

- 3 showcase posts
- 2 how-to posts
- 1 creator spotlight
- 1 challenge prompt
- 1 referral/mission education post

Daily check:

- new signup
- first posts
- top creators
- referral activations
- support/report issues

## Marketing Questions Before Building Feature

1. Feature นี้ช่วย acquisition, activation หรือ retention?
2. อธิบายใน 1 social post ได้ไหม?
3. ทำให้ referral loop ดีขึ้นไหม?
4. มี abuse path ไหม?
5. วัดผลได้ด้วย event อะไร?
