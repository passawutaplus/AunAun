# Aplus1 Copy And Language System

Purpose: make Aplus1 feel like an opportunity platform, not just a job board or portfolio host.

## Core Language Decision

Use **"โอกาส"** as the top-level word.

Use **"งาน"** only when the context is specifically paid work, job posts, employment, or freelance project execution.

Why:

- "โอกาส" includes paid and non-paid value.
- It feels warmer and more creator-first.
- It fits students, emerging creators, collaborations, internships, mentorship, exposure, and future contact.
- It separates Aplus1 from Fastwork/Fiverr/JobsDB framing.

## Brand Core

| Layer | Copy |
|---|---|
| Core phrase | ผลงาน -> โอกาส |
| Founder sentence | ให้ผลงานจริงพาครีเอเตอร์ไปเจอโอกาสใหม่ |
| Creator promise | ลงผลงานให้คนเห็นศักยภาพ และเปิดรับโอกาสที่ใช่ |
| Hirer promise | ค้นหาครีเอเตอร์จากผลงานจริง ก่อนเริ่มคุยโอกาส |
| Product angle | ทุกผลงานควรเป็นประตูสู่โอกาส ไม่ใช่แค่รูปในพอร์ต |

## Positioning Statements

### Option 1: Broad Founder Positioning

For Thai creative people who want their work to lead somewhere, Aplus1 is an opportunity platform that turns real projects into discovery, trust, and new conversations, unlike job boards or portfolio sites that stop at applications or showcases.

Thai:

> Aplus1 คือแพลตฟอร์มที่ให้ผลงานจริงพาครีเอเตอร์ไทยไปเจอโอกาสใหม่ ไม่ว่าจะเป็นงานจ้าง การร่วมโปรเจกต์ ฝึกงาน เข้าทีม หรือการถูกค้นพบ

### Option 2: Creator-First

> พื้นที่สำหรับครีเอเตอร์ที่อยากให้ผลงานพูดแทนตัวเอง และพาไปเจอโอกาสที่ใช่

### Option 3: Hirer-First

> ค้นหาคนจากผลงานจริง เห็นสไตล์ เห็นศักยภาพ แล้วค่อยเริ่มคุยโอกาส

### Option 4: Student/Emerging Creator

> เริ่มจากผลงานชิ้นเดียว ให้คนเห็นว่าคุณทำอะไรได้

### Option 5: Anti-Overpromise / Trust

> ลดการเดาจากคำขายตัวเอง ด้วยผลงานจริง บทบาทจริง และบริบทของงานจริง

## Headline Bank

### Main Hero Candidates

1. ผลงานจริง พาไปเจอโอกาสใหม่
2. ให้ผลงานพาคุณไปเจอโอกาสที่ใช่
3. เปิดพื้นที่ให้ผลงานของคุณถูกค้นพบ
4. ค้นหาครีเอเตอร์จากผลงานจริง
5. โปรไฟล์เดียว เชื่อมต่อทุกโอกาสของครีเอทีฟ

Recommended MVP hero:

> ผลงานจริง พาไปเจอโอกาสใหม่

Reason:

- Short.
- Clear.
- Works for creators and hirers.
- Keeps "โอกาส" central.

### Subheadline Candidates

1. ลงผลงานพร้อมบริบท เปิดรับโอกาสที่ใช่ และให้คนติดต่อจากงานที่เขาสนใจจริง ๆ
2. Aplus1 ช่วยให้ครีเอเตอร์โชว์ศักยภาพผ่านผลงานจริง และช่วยผู้จ้างค้นหาคนจากสไตล์ บทบาท และคุณภาพของงาน
3. ไม่ใช่แค่เว็บพอร์ต ไม่ใช่แค่เว็บหางาน แต่คือพื้นที่ที่ผลงานเชื่อมต่อคนกับโอกาส
4. ดูงาน เข้าใจบทบาท แล้วค่อยเริ่มคุยโอกาส

Recommended:

> ลงผลงานพร้อมบริบท เปิดรับโอกาสที่ใช่ และให้คนติดต่อจากงานที่เขาสนใจจริง ๆ

## CTA System

### Creator CTA

| Context | Primary CTA | Secondary |
|---|---|---|
| landing | เริ่มลงผลงาน | ดูตัวอย่างโปรไฟล์ |
| empty profile | เพิ่มผลงานแรก | ตั้งค่าโอกาส |
| project upload | เผยแพร่ผลงาน | บันทึกฉบับร่าง |
| opportunity status | เปิดรับโอกาส | เลือกประเภทโอกาส |
| encouragement | ให้ผลงานชิ้นนี้พาคุณไปเจอโอกาส | เพิ่มบริบทให้งาน |

### Hirer CTA

| Context | Primary CTA | Secondary |
|---|---|---|
| project card | ดูผลงาน | เก็บไว้ |
| project detail | คุยต่อจากผลงานนี้ | ดูโปรไฟล์ |
| creator profile | ชวนคุยโอกาส | เก็บ creator |
| shortlist | ติดต่อจากงานที่สนใจ | เปรียบเทียบ |
| unsure | เก็บไว้ติดต่อภายหลัง | แชร์ให้ทีม |

### Avoid As Primary CTA

- Hire
- จ้างเลย
- ซื้อแพ็กเกจ
- สมัครงานทันที

These are too transactional for the core opportunity thesis. They can exist in paid-work-specific flows, but should not dominate discovery.

## Profile Status Language

Recommended statuses:

| Status | Meaning | Use Case |
|---|---|---|
| เปิดรับโอกาส | broad default | creator is open to possibilities |
| รับงานจ้าง | paid project | freelancer/paid work |
| พร้อมร่วมโปรเจกต์ | collaboration | peer/studio collab |
| มองหาฝึกงาน | internship | student/new grad |
| สนใจเข้าทีม | full-time/part-time | employment |
| เปิดรับ feedback / mentor | learning | junior/student |
| ยังไม่รับงาน แต่คุยโอกาสได้ | soft availability | experienced creator |

Default recommendation:

> เปิดรับโอกาส

Then let users choose opportunity types below it.

## Onboarding Copy

### Creator Onboarding

Step 1:

> เริ่มจากบอกว่าอยากเปิดรับโอกาสแบบไหน

Step 2:

> เพิ่มผลงานที่อยากให้คนจดจำ

Step 3:

> ใส่บริบทสั้น ๆ เพื่อให้คนเข้าใจศักยภาพของคุณ

Step 4:

> เผยแพร่ แล้วให้ผลงานเริ่มพาคุณไปเจอโอกาส

### Hirer Onboarding

Step 1:

> เริ่มจากดูงานที่ใกล้โจทย์

Step 2:

> ดูบทบาทและวิธีคิดของ creator

Step 3:

> เก็บไว้หรือคุยต่อจากผลงานนั้นได้เลย

## Trust Copy For Hirers

- ดูจากผลงานจริง ไม่ใช่แค่คำอธิบาย
- เห็นบทบาทและ process ก่อนเริ่มคุย
- ติดต่อจากงานที่คุณสนใจ เพื่อให้ brief ชัดตั้งแต่แรก
- เก็บ creator ไว้เปรียบเทียบก่อนตัดสินใจ
- เริ่มจากบทสนทนา ไม่จำเป็นต้องจ้างทันที

## Encouragement Copy For Creators

- งานเรียนก็เป็นจุดเริ่มต้นของโอกาสได้ ถ้าเล่าให้เห็นโจทย์และบทบาท
- ไม่ต้องมีพอร์ตใหญ่ เริ่มจากงานที่บอกตัวตนของคุณได้ดีที่สุด
- คนไม่ได้ดูแค่ความสวย เขาดูวิธีคิดและศักยภาพด้วย
- เปิดรับโอกาสได้หลายแบบ ไม่จำเป็นต้องรับงานจ้างอย่างเดียว
- ให้ผลงานช่วยคัดคนที่สนใจงานแนวเดียวกับคุณ

- ศักยภาพ
- เปิดรับ

- บทบาท
- วิธีคิด
- สไตล์
- งานแนวนี้
- เก็บไว้ติดต่อภายหลัง
- ถูกค้นพบ
- ร่วมโปรเจกต์

## Words To Avoid Or Use Carefully

| Word | Why |
|---|---|
| สมัครงาน | too job-board focused |
| จ้างเลย | too transactional / high pressure |
| ถูกที่สุด | wrong strategic frame |
| แพ็กเกจ | Fastwork/Fiverr frame |
| มืออาชีพเท่านั้น | scares students/new creators |
| งานเทพ | vague and intimidating |
| viral | distracts from opportunity quality |
| ranking | may create anxiety and status games |

## Thai / English Tone

Default should be Thai-first with selective English for common professional terms.

Use:

- project
- brief
- portfolio
- creator
- freelance
- collaboration
- case study

But avoid inconsistent button text like "Close" or "Invite to Job" when surrounding UI is Thai.

## Copy Test Recommendations

Test these 5:

1. ผลงานจริง พาไปเจอโอกาสใหม่
2. ให้ผลงานพาคุณไปเจอโอกาสที่ใช่
3. ดูงานก่อน ค่อยจ้างคน
4. ค้นหาครีเอเตอร์จากผลงานจริง
5. โปรไฟล์เดียว เชื่อมต่อทุกโอกาสของครีเอทีฟ

Expected:

- Creators may prefer 1 or 2.
- Hirers may prefer 3 or 4.
- Brand-level homepage may use 1, with hirer sections using 4.

## Final Recommendation

Use a two-sided language system:

For creators:

> ให้ผลงานพาคุณไปเจอโอกาสที่ใช่

For hirers:

> ค้นหาครีเอเตอร์จากผลงานจริง

For the brand:

> ผลงานจริง พาไปเจอโอกาสใหม่
