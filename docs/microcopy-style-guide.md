# Microcopy Style Guide — So1o + 1PX Ecosystem

คู่มือเขียนข้อความในแอปให้กระชับ อ่านง่าย ไม่ซ้ำซ้อน

---

## หลักการ 6 ข้อ

| หลักการ | กฎ |
|---------|-----|
| หนึ่งข้อความ หนึ่งหน้าที่ | title = สรุป · desc = รายละเอียด · อย่าพูดเรื่องเดียวกัน 3 ชั้น |
| ตัดคำซ้ำ | ประโยคหลังอธิบายสิ่งที่ประโยคแรกบอกแล้ว → ลบ |
| ไทยเป็นหลัก | UI ผู้ใช้ไทย: ไทยเต็ม · อังกฤษเฉพาะชื่อเทคนิค/แบรนด์ |
| ความยาว | แบนเนอร์ ≤ 1 ประโยค · bullet ≤ 14 คำ · disclaimer 1 ประโยค |
| น้ำเสียง | กันเอง มืออาชีพ ไม่ทางการเกิน — เหมาะกับฟรีแลนซ์ไทย |
| Disclaimer | แสดงครั้งเดียวต่อ context — ไม่ซ้ำใน guide + tip + help |

## ข้อความมาตรฐาน (import จาก `copyConstants.ts`)

| ใช้เมื่อ | ข้อความ |
|---------|---------|
| ภาษี/ราคา | `คำแนะนำเบื้องต้น — ดูหน้างานจริงก่อนตัดสินใจ` |
| ภาษีประมาณการ | `ค่าที่แสดงเป็นประมาณการ — ปรึกษานักบัญชีก่อนยื่นจริง` |
| กฎหมาย | `คำแนะนำเบื้องต้น ไม่ใช่คำปรึกษาทางกฎหมาย` |
| Demo (So1o) | `โหมดทดสอบ — บันทึกได้จริง ชำระเงินเป็น sandbox` |
| Demo (1PX) | `โหมด UX Research — บัญชี demo บันทึกถาวร` |

## คำที่ควรหลีกเลี่ยง → คำที่แนะนำ

| หลีกเลี่ยง | แนะนำ |
|-----------|-------|
| บันทึกข้อมูลจริงลงฐานข้อมูล cloud | บันทึกถาวร / บันทึกได้จริง |
| เว็บเชื่อม Supabase จริง ไม่ใช่ mockup | บัญชี demo บันทึกถาวร |
| นี่เป็นเพียงคำแนะนำเบื้องต้น โปรดพิจารณาหน้างานจริงอีกครั้งนะครับ | คำแนะนำเบื้องต้น — ดูหน้างานจริงก่อนตัดสินใจ |
| หน้านี้ไม่มีอยู่จริง + ลิงก์อาจพิมพ์ผิด หน้าถูกย้าย | เลือกอย่างใดอย่างหนึ่ง (title หรือ desc) |
| ทุกฟรีแลนซ์คือ 1 PX — รวมกันเป็นภาพใหญ่... (ซ้ำ tagline) | แสดง tagline อย่างเดียวต่อหน้า |

## ตัวอย่าง Before → After

### Demo banner (So1o)

- **Before:** ล็อกอินและบันทึกข้อมูลได้จริง (Supabase) · ใช้บัญชีทดสอบของตัวเอง · การชำระเงินเป็นโหมดทดสอบ
- **After:** โหมดทดสอบ — บันทึกได้จริง ชำระเงินเป็น sandbox

### Research warning (1PX)

- **Before:** เว็บนี้เชื่อมฐานข้อมูลจริง — ล็อกอินด้วยบัญชี demo แล้วแก้โปรไฟล์/โพสต์/คอมเมนต์บันทึกถาวร (ใช้ร่วมกับนักวิจัยคนอื่น)
- **After:** บัญชี demo บันทึกถาวร — ใช้ร่วมกัน อย่าสมัครใหม่

### Error 404

- **Before:** tagline + title + desc ทั้งหมดบอกว่า "หน้าไม่มี"
- **After:** title = ข้อความหลัก · desc = สาเหตุ + ทางออก 1 ประโยค · EN เป็น subtitle เล็ก
